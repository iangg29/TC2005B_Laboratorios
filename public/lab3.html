Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque puede ocasionar que el texto o contenido de la página sea difícil de ver o leer.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Dependiendo de la estructura del sitio web, normalmente el usar % facilita el hacer el diseño responsive, sin embargo el uso de px y pt también es importante.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque el archivo raw de CSS cuenta con muchos espacios y saltos de línea. Y en una versión minimizada, esto se elimina, por lo tanto el navegador lo puede leer y ejecutar de una manera más óptima y rápido en la página web.
